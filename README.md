# WinServer-Manager
WinServer Manager 是一款專為 Windows 環境設計的輕量化伺服器管控解決方案。透過單一集中式儀表板，使用者能即時掌握主機的系統資源負載（CPU、記憶體、GPU）以及各個伺服器實例的運行狀態。  核心功能亮點：  即時監控中心： 視覺化呈現全局資源負載與各伺服器實例的運作效能。  集中化管理： 透過「新增」功能輕鬆部署並管理多個伺服器實例。  智慧維運看板： 整合運行狀態統計與「看門狗守護」機制，確保服務穩定不中斷。  全域日誌追蹤： 自動記錄系統事件與操作日誌，便於問題排查與狀態追蹤。  這套系統適合需要高效監控 Windows 伺服器資源，並追求直覺化管理介面的使用者。
